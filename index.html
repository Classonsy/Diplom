Обзор системы

Система управления персоналом для сети магазинов "Красное&Белое" представляет собой комплексное решение на базе платформы 1С:Предприятие. Система предназначена для автоматизации процессов работы с персоналом, включая:

  • Кадровый учет
  • Расчет заработной платы
  • Планирование графиков работы
  • Управление эффективностью
  • Обучение и развитие персонала
  • Рекрутинг и адаптацию

Структура базы данных

Справочники

  • Сотрудники - данные о персонале
  • Подразделения - филиалы и магазины
  • Должности - должностные позиции
  • ГрафикиРаботы - шаблоны графиков
  • НаборыНавыков - требуемые компетенции

Документы

  • ПриемНаРаботу - оформление сотрудника
  • Увольнение - прекращение трудовых отношений
  • КадровыйПеревод - изменение должности/подразделения
  • ТабельУчетаРабочегоВремени - учет отработанного времени
  • НачислениеЗарплаты - расчет оплаты труда
  • Больничный - учет больничных листов
  • Отпуск - оформление отпусков
  • ОценкаПроизводительности - KPI сотрудников

Регистры

  • ШтатноеРасписание - учет штатных единиц
  • ГрафикиРаботыСотрудников - индивидуальные графики
  • НавыкиСотрудников - учет компетенций
  • ВыполнениеПланов - показатели эффективности

Рабочие процессы

Наем сотрудника

  1. Создание заявки на подбор персонала
  2. Проведение собеседований и отбора
  3. Оформление документа "ПриемНаРаботу"
  4. Назначение графика работы
  5. Определение программы адаптации

Учет рабочего времени

  1. Планирование графиков на месяц
  2. Ежедневное отражение фактической явки
  3. Учет отклонений (опоздания, ранние уходы)
  4. Формирование табеля учета рабочего времени
  5. Расчет показателей соблюдения дисциплины

Управление эффективностью

  1. Установка плановых показателей
  2. Сбор данных о выполнении
  3. Проведение аттестаций
  4. Расчет KPI и формирование премий
  5. Планирование развития

Отчеты

  • Штатное расписание - текущая структура и вакансии
  • Анализ текучести кадров - динамика увольнений по причинам
  • Производительность персонала - KPI по сотрудникам и магазинам
  • Фонд оплаты труда - анализ затрат на персонал
  • Анализ рабочего времени - эффективность использования трудовых ресурсов
  • Карта компетенций - обеспеченность навыками по подразделениям

Интеграция с другими системами

Система управления персоналом интегрирована со следующими системами:

  • 1С:Бухгалтерия - для учета расходов на оплату труда
  • 1С:Розница - для получения данных о продажах и KPI
  • Система контроля доступа - для учета фактического времени присутствия
  • Корпоративный портал - для доступа сотрудников к персональной информации
